# 对比试验索引

这个文档用于记录每次对比试验：**试验名称**、**要解决的问题**、**数据范围/口径**、**关键配置/命令**、**结果与结论**。

约定：
- 每个对比试验在 `docs/对比试验/<试验名称>/` 下建立子文件夹。
- 子文件夹内至少包含一个 `报告.md`（可附加更多运行记录/数据说明）。
- `docs/` 根目录只保留“入口/跳转”类文档。

说明：为方便阅读与固定排序，子文件夹采用 `NN_<试验名称>/` 的编号前缀（例如 `01_组件来源对比/`）。
